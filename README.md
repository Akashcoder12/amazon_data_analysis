# 🍽️ Zomato Data Analysis using Python

## 📌 Project Overview

This project explores Zomato restaurant data using Python to understand customer preferences, restaurant services, and pricing trends.

The objective is to perform Exploratory Data Analysis (EDA) by cleaning the dataset, visualizing important patterns, and extracting meaningful business insights that can help restaurants make data-driven decisions.

---

## 🎯 Business Problems

The analysis aims to answer the following questions:

1. Do more restaurants provide **online delivery** compared to **offline services**?

2. Which **types of restaurants** are most preferred by customers?

3. What **price range** do couples generally prefer for dining out?

---

## 📂 Dataset

The dataset contains information about restaurants listed on Zomato, including:

- Restaurant Name
- Restaurant Type
- Online Order Availability
- Table Booking
- Votes
- Ratings
- Cost for Two


Dataset file:

```
Zomato_data.csv
```
